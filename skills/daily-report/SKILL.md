# 日报撰写 Skill

> 快速生成当日工作日报。

## 启动指令

在 Cursor 中输入 `/写日报`

## 工作流程

1. AI 回顾今天的对话历史和文件变更
2. 提炼关键工作内容和产出
3. 生成日报草稿
4. 你审核调整后提交

## 日报格式

```markdown
# 日报 — {日期}

## 今日完成
- [ ] 任务1：具体描述 + 产出物
- [ ] 任务2：具体描述 + 产出物

## 遇到的问题
- 问题描述 + 当前状态

## 明日计划
- 计划1
- 计划2

## 个人思考（可选）
- 行业观察 / 技能提升 / 想法
```
