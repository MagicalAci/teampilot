# 竞品深度调研 Skill

> 本 Skill 将通过苏格拉底式引导，帮助你完成一份专业级的竞品调研报告。

## 启动指令

在 Cursor 中输入 `/开始调研`，AI 将引导你逐步完成以下环节：

## 工作流程

### Phase 1: 理解需求（10-15min）
- AI 阅读 `brief.md`，提炼负责人的核心关注点
- 与你确认调研范围、重点维度、竞品选择

### Phase 2: 背景研究（30-45min）
- AI 搜索产品基本信息、融资背景、团队情况
- 你补充个人认知和行业理解

### Phase 3: 产品体验（1-2h）
- 逐个体验竞品核心功能
- 截图记录关键界面和交互
- AI 帮你组织截图和观察笔记

### Phase 4: 深度分析（1-1.5h）
- AI 能力测试（如适用）
- 商业模式 / 定价策略分析
- 用户口碑采集（小红书、知乎等）

### Phase 5: 报告撰写（1-1.5h）
- AI 基于收集的素材生成报告框架
- 你审核、补充见解、调整结论
- 生成功能对比矩阵和 SWOT 分析

### Phase 6: 自查提交
- AI 按 brief.md 中的要求逐条核查
- 确认无遗漏后提交 PR

## 交付物要求

- `deliverables/research-report.md` — 主报告
- `deliverables/screenshots/` — 产品截图
- `deliverables/comparison-matrix.md` — 功能对比表（可选）

## 质量标准

- 每个竞品至少覆盖：功能、体验、定价、AI能力、用户评价 5 个维度
- 数据有来源标注
- 结论有论据支撑
- SWOT 分析 Threats 要具体，不能泛泛而谈
