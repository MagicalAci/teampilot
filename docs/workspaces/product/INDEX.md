## 本文档回答的核心问题

1. “真题去学”和“AI 真题组卷”是否应该做成同一个网站。
2. 这个网站首版应该服务谁、解决什么问题、怎么搭建。
3. 哪些能力要先做，哪些能力应明确不做。

## 当前结论

- 应做成同一网站下的双入口产品，而不是两个彼此割裂的站点。
- 网站的核心定位不是“更多课程”或“更大题库”，而是“用真题带学习，用 AI 扩练习”。
- 首版要先把学习闭环做通：`真题 -> 知识点 -> 变式练习 -> 小测 -> 清单`，以及 `薄弱点 -> AI 组卷 -> 限时练 -> 复盘`。
- 首版不建议做社区、直播课、拍照搜题和泛题库平台。
